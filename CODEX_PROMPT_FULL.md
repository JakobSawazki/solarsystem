# Codex-Startprompt – Simuliertes Sonnensystem

Arbeite in diesem Repository als professioneller Frontend-/WebGL-Entwickler. Lies zuerst `AGENTS.md`, `SKILL.md`, `TASKS.md`, `docs/PROJECT_SPEC.md` und `docs/DATA_MODEL.md`. Danach bearbeitest du ausschließlich die erste Task mit Status `OPEN` in `TASKS.md`.

Ziel des Projekts ist eine hochwertige interaktive Homepage über unser Sonnensystem, veröffentlichbar über GitHub Pages. Die Anwendung soll mit Vite und Three.js umgesetzt werden. Sie soll eine dreh- und zoombare 3D-Ansicht mit Sonne, acht Planeten, Orbitlinien, Daten-Infopanel, Maßstabsmodi und modernen visuellen Effekten enthalten.

Wichtige Produktentscheidung: Ein einziger „echter“ Maßstab für Größen und Entfernungen ist nicht sinnvoll bedienbar. Implementiere daher mehrere klar beschriftete Modi: cinematic/hybrid, Größenmaßstab, Entfernungsmaßstab und Vergleichsmodus.

Arbeitsweise:

- Nur eine Task gleichzeitig bearbeiten.
- Keine große unstrukturierte Komplettumsetzung ohne Dokumentation.
- Vor Änderungen kurz prüfen, ob die vorhandene Struktur sinnvoll ist.
- Nach jeder erledigten Task `TASKS.md`, `docs/VERSION_STATE.md` und `docs/CHANGELOG.md` aktualisieren.
- App muss lokal mit `npm install`, `npm run dev` und `npm run build` funktionieren.
- GitHub-Pages-Fähigkeit muss erhalten bleiben.
- Bei UI-Texten Deutsch verwenden.
- Performance, Mobile-Bedienung und Barrierefreiheit berücksichtigen.

Beginne mit der ersten offenen Task.
