# Akzeptanztests – Simuliertes Sonnensystem

## Build und Start

- [ ] `npm install` läuft ohne kritischen Fehler.
- [ ] `npm run dev` startet die App.
- [ ] `npm run build` läuft erfolgreich.
- [ ] `npm run preview` zeigt die gebaute App.

## 3D-Szene

- [ ] Sonne ist sichtbar.
- [ ] Alle acht Planeten sind sichtbar oder über UI fokussierbar.
- [ ] Ansicht lässt sich drehen.
- [ ] Ansicht lässt sich zoomen.
- [ ] Fenstergrößenänderung verzerrt die Szene nicht.

## Maßstabsmodi

- [ ] Cinematic Mode vorhanden und als visuell optimiert erklärt.
- [ ] Größenmaßstab vorhanden und erklärt.
- [ ] Entfernungsmaßstab vorhanden und erklärt.
- [ ] Vergleichsmodus vorhanden oder als klarer nächster Schritt dokumentiert.
- [ ] Kein UI-Text behauptet fälschlich, alle Größen und Entfernungen seien gleichzeitig vollständig maßstabsgetreu.

## Infopanel

- [ ] Klick/Tap auf Planet öffnet Infopanel.
- [ ] Objektliste fokussiert Planeten.
- [ ] Infopanel zeigt Daten aus JSON.
- [ ] `Esc` schließt Panel, sofern implementiert.

## Animation

- [ ] Planeten bewegen sich optional auf Bahnen.
- [ ] Pause/Zeitregler funktioniert, sofern implementiert.
- [ ] Animation bleibt flüssig.

## Mobile/Responsive

- [ ] App ist auf Smartphone nutzbar.
- [ ] UI-Elemente überdecken die Szene nicht unbrauchbar.
- [ ] Texte sind lesbar.

## GitHub Pages

- [ ] Workflow läuft unter Actions erfolgreich.
- [ ] Veröffentlichtes GitHub-Pages-Ergebnis lädt JS/CSS korrekt.
- [ ] JSON-Daten werden auch im GitHub-Pages-Unterpfad geladen.
